# Secure Password Manager in C++

## 📌 Overview
This project implements a command-line password manager using C++ that allows users to securely store, retrieve, search, and delete credentials. It uses file handling for persistent storage and applies basic encryption techniques.

## 🚀 Features
- Add new credentials (website, username, password)
- View stored credentials
- Search for specific entries
- Delete credentials
- Master password authentication
- Basic encryption for stored data

## 🛠️ Technologies Used
- C++ Programming
- Object-Oriented Programming (OOP)
- File Handling (ifstream, ofstream)
- String Manipulation

## ⚙️ How It Works
- User authenticates using a master password
- Data is stored in file with simple encryption
- Credentials are retrieved and decrypted when accessed
